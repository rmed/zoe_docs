Installation
============

.. _Zoe startup kit: https://github.com/voiser/zoe-startup-kit

Fetch the source for the `Zoe startup kit`_ and place it in the directory you want to run *Zoe* from, such as ``/home/zoe``. Navigate to that directory and make the following scripts executable::

    $ chmod +x zoe.sh

::

    $ chmod +x etc/environment.sh
